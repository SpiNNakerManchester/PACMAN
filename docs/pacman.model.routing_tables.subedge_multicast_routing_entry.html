add_subedge(subedge, processor_ids, link_ids)[source]¶

Adds a subedge to the entry adding in the processor_ids and link_ids that are the destination of that subedge. The pre_subvertex of the subedge must be the same as that for the current subedges.

Parameters:
  • subedge (pacman.model.subgraph.subedge.Subedge) – The subedge to add
  • processor_ids (iterable of int) – The processor ids that are the destination for the given subedge
  • link_ids (iterable of int) – The link ids that are the destination for the given subedge
Returns:

Nothing is returned
Return type:

None
Raises pacman.exceptions.PacmanInvalidParameterException:
 

If the subedge does not have the same pre_subvertex as the existing subedges
