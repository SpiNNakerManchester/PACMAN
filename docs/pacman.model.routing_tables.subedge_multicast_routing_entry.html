pacman.model.routing_tables.subedge_multicast_routing_entry module¶

class pacman.model.routing_tables.subedge_multicast_routing_entry.SubedgeMulticastRoutingEntry(subedge_info, destination_route, source_link_id)[source]¶

Bases: object

Represents a routing entry for a specific subedge of a subgraph. Note that multiple entries may be merged to create single MulticastRoutingEntry

Parameters:
  • subedge_info (pacman.model.routing_info.subedge_routing_info.SubedgeRoutingInfo) – the routing information for a specific subedge
  • destination_route (pacman.model.routing_tables.multicast_route.MulticastRoute) – the destination route for the subedge
  • source_link_id (int) – The id of the link on the source chip from which the traffic will be received
Raises None:

does not raise any known exceptions

Attributes

destination_route The route to send received packets down
source_link_id The link on the source chip down which the packet was sent
subedge_info The key and mask in the routing entry
