pacman.model.placements.placement module¶

class pacman.model.placements.placement.Placement(subvertex, x, y, p)[source]¶

Bases: object

Represents a placement of a subvertex on a specific processor on a specific chip in the machine

Parameters:
  • subvertex (pacman.model.subgraph.subvertex.Subvertex) – The subvertex that has been placed
  • x (int) – the x-oordinate of the chip on which the subvertex is placed
  • y (int) – the y-coordinate of the chip on which the subvertex is placed
  • p (int) – the id of the processor on which the subvertex is placed
Raises None:

does not raise any known exceptions

Attributes

p The id of the processor of the chip where the subvertex is placed
subvertex The subvertex that was placed
x The x-coordinate of the chip where the subvertex is placed
y The y-coordinate of the chip where the subvertex is placed
