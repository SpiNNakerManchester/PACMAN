pacman.model.routing_info.subedge_routing_info module¶

class pacman.model.routing_info.subedge_routing_info.SubedgeRoutingInfo(subedge, key, mask)[source]¶

Bases: object

Associates a subedge to its routing information (key and mask)

Parameters:
  • subedge (pacman.model.subgraph.subedge.Subedge) – the subedge down which the routing key and mask will be used
  • key (int) – the key which will be sent by the subvertex at the start of the subedge
  • mask (int) – the mask of the key which indicates which bits in the key are used, and which can be ignored
Raises None:

does not raise any known exceptions

Attributes

key The routing key
mask The mask of the routing key
subedge The subedge which the information is about
