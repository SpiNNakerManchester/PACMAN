pacman.model.graph.graph module¶

class pacman.model.graph.graph.Graph(label=None, vertices=None, edges=None)[source]¶

Bases: object

Represents a collection of vertices and edges between vertices that make up a graph

Parameters:
  • label (str) – An identifier for the graph
  • vertices (iterable of pacman.model.graph.vertex.Vertex) – An iterable of initial vertices in the graph
  • edges (iterable of pacman.model.graph.edge.Edge) – An iterable of initial edges in the graph
Raises pacman.exceptions.PacmanInvalidParameterException:
 

  • If one of the edges is not valid
  • If one of the vertices in not valid
