PACMAN¶

Provides various functions which together can be used to take a graph and split it into pieces that can be loaded on to a machine, along with routes between the pieces. The main items provided are:

  • pacman.operations.partitioner.Partitioner
  • pacman.operations.placer.Placer
  • pacman.operations.pruner.Pruner
  • pacman.operations.routing_info_allocator.RoutingInfoAllocator
  • pacman.operations.router.Router
