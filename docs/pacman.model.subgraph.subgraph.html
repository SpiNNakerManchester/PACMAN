pacman.model.subgraph.subgraph module¶

class pacman.model.subgraph.subgraph.Subgraph(graph, label=None, subvertices=None, subedges=None)[source]¶

Bases: object

Represents a partitioning of a graph

Parameters:
  • graph (pacman.model.graph.graph.Graph) – the graph of which this is a subgraph
  • label (str) – an identifier for the subgraph
  • subvertices (iterable of pacman.model.subgraph.subvertex.Subvertex) – an iterable of vertices in the graph
  • subedges (iterable of pacman.model.subgraph.subedge.Subedge) – an iterable of subedges in the graph
Raises pacman.exceptions.PacmanInvalidParameterException:
 

  • If one of the subedges is not valid
  • If one of the subvertices is not valid

Attributes

graph The graph of which this is a subgraph
label The label of the subgraph
