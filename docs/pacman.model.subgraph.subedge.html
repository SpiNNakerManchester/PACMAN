pacman.model.subgraph.subedge module¶

class pacman.model.subgraph.subedge.Subedge(edge, pre_subvertex, post_subvertex, label=None)[source]¶

Bases: object

Represents part of a division of an edge to match the division of the vertices on either side of the edge

Parameters:
  • edge (pacman.model.graph.edge.Edge) – The edge which this is a subedge of
  • pre_subvertex (pacman.model.subgraph.subvertex.Subvertex) – the subvertex at the start of the subedge
  • post_subvertex (pacman.model.subgraph.subvertex.Subvertex) – the subvertex at the end of the subedge
  • label (str) – The name of the edge
Raises None:

Raises no known exceptions

Attributes

label The label of the subedge
