pacman.model.machine.chip module¶

class pacman.model.machine.chip.Chip(x, y, processors, router, sdram, ip_address=None)[source]¶

Bases: object

Represents a chip with a number of cores, an amount of SDRAM shared between the cores, and a router

Parameters:
  • x (int) – the x-coordinate of the chip’s position in the two-dimentional grid of chips
  • y (int) – the y-coordinate of the chip’s position in the two-dimentional grid of chips
  • processors (iterable of pacman.model.machine.processor.Processor) – an iterable of processor objects
  • router (pacman.model.machine.router.Router) – a router for the chip
  • sdram (pacman.model.machine.sdram.SDRAM) – an SDRAM for the chip
  • ip_address (str) – the IP address of the chip or None if no ethernet attached
Raises pacman.exceptions.PacmanAlreadyExistsException:
 

If processors contains any two processors with the same processor_id

Attributes

ip_address The IP address of the chip
processors An iterable of available processors
router The router object associated with the chip
sdram The sdram associated with the chip
x The x-coordinate of the chip in the two-dimensional grid of chips
y The y-coordinate of the chip in the two-dimensional grid of chips

Methods
