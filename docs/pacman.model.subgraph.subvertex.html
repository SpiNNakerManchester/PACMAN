pacman.model.subgraph.subvertex module¶

class pacman.model.subgraph.subvertex.Subvertex(vertex, lo_atom, hi_atom, label=None, constraints=None)[source]¶

Bases: object

Represents a sub-set of atoms from a Vertex

Parameters:
  • vertex (pacman.model.graph.vertex.Vertex) – The vertex which this is a subvertex of
  • lo_atom (int) – The id of the first atom in the subvertex with reference to the atoms in the vertex
  • hi_atom (int) – The id of the last atom in the subvertex with reference to the atoms in the vertex
  • label (str) – The name of the subvertex, or None if no name
  • constraints (iterable of pacman.model.constraints.abstract_constraint.AbstractConstraint) – The constraints of the subvertex
Raises pacman.exceptions.PacmanInvalidParameterException:
 

  • If one of the constraints is not valid
  • If lo_atom is less than 0
  • If hi_atom is less than lo_atom

Attributes

constraints An iterable constraints for the subvertex
hi_atom The id of the last atom in the subvertex
label The label of the subvertex
lo_atom The id of the first atom in the subvertex
n_atoms The number of atoms in the subvertex
vertex The vertex object to which the subvertex refers

Methods
